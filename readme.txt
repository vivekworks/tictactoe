Please follow the steps
----------------------------
1. Download the file and extract it on to a folder.
2. Press Ctrl+Shift+Rightclick on the empty space of the folder and choose "open command window here" (or in Windows 10 and above, "open powershell window here"). Alternatively, open command window and move to the directory where the files are extracted in it.
3. In the window, enter as below and press enter.
	  java -jar tictactoe.jar
4. Play the game and enjoy!
